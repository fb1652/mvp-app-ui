# mri-app-ui
Angular 5 UI for the MRI Application
### Prereqs
* Docker
### Install
```
npm install
```
### Start
```
npm start
```
### Stop
```
npm stop
```
### Authorize Users
List, add and revoke users.
```
npm run auth -- list
npm run auth -- add <email> [USER|ADMIN]
npm run auth -- revoke <email>
```
