Download and Install the MRI Toolkit

After you have downloaded the MRI Toolkit file, follow these steps to install it:

Extract the following files from the CA-MRI-Toolkit.zip file:
  • CA.MRI.LOAD.xmi - The CA MRI LOAD library in xmit format
  • CA.MRI.CNTL.xmi - The CA MRI CNTL library in xmit format
  • CA MRI Toolkit pdf - The documentation for the CA MRI Toolkit
 Upload and install the CA MRI Toolkit xmit files on DASD that is accessible to the LPARs where you intend to run the MRI scans.

Note: If you have previously installed the CA MRI Toolkit, create a backup of your existing CA.MRI.LOAD and CA.MRI.CNTL load libraries.

A. From your PC, use FTP to upload the CA.MRI.LOAD.xmi and CA.MRI.CNTL.xmi files to your LPAR.

    ftp xxx.xxx.com
    bin
    quote site CYLINDERS PRIMARY=5 FILETYPE=SEQ RECFM=FB LRECL=80
    put CA.MRI.LOAD.xmi
    quote site CYLINDERS PRIMARY=1 FILETYPE=SEQ RECFM=FB LRECL=80
    put CA.MRI.CNTL.xmi
    quit

B. On your LPAR, receive the uploaded xmit files. Log onto TSO on an LPAR that has access to the uploaded xmit files.

    1. Start ISPF, and navigate to ISPF Option 6 (the ISPF Command Shell)

    2. On the command prompt enter:

        RECEIVE INDA('your uploaded CA MRI LOAD xmit dataset name')

    then enter:

        RECEIVE INDA('your uploaded CA MRI CNTL xmit dataset name')
